Preguntas Frecuentes

¿El registro tiene algún costo?

No, la participación en Liquid.AI es completamente gratuita. Solo necesitas una cuenta de GitHub para registrarte.

¿Puedo participar individualmente?

Sí, puedes participar solo o en equipos de hasta 4 integrantes. Te ayudaremos a buscar equipo si no tienes uno.

Cualquier API o modelo de IA es válido. Brindaremos accesos premium exclusivos de nuestros patrocinadores.

¿Cómo se realiza el despliegue?

Se sugiere que todas las interfaces sean desplegadas en plataformas web públicas para facilitar su testeo directo.

¿Listo para cambiar las reglas del juego?

Las plazas son limitadas para garantizar la mejor calidad en la mentoría interactiva. Asegura tu lugar.
